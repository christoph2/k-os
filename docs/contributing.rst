Contributing
============

Authors
-------
See the top-level ``AUTHORS`` file for the list of contributors. The primary maintainer
is Christoph Schueler.

How to contribute
-----------------
- Open issues with reproducible steps and a short description of the problem.
- Submit pull requests with focused changes and a clear description of intent.
- State the target platform or port when opening platform-specific changes.

Guidelines
----------
- Follow repository coding conventions and module structure.
- Provide sample OIL configurations or test cases where appropriate.
- Keep changes small and well-documented.
